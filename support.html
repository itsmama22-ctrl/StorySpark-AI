Common Questions

How do I create a story?
You can create stories in two ways:
  • Write Your Own: Go to the Home screen, tap "Create New Story", write your story text, and tap "Generate" to create illustrations.
  • AI Generation: Switch to "Generate Story" mode, enter a prompt (e.g., "A story about a brave dragon"), and let AI create the full story for you.
Why do I need to watch an ad to generate stories?
StorySpark AI uses Google AdMob to keep the app free. Rewarded ads appear when you want to:
  • Generate a full AI story
  • Use premium illustration styles
  • Export your story as PDF
  • Regenerate a page
Watching a short ad unlocks these premium features and helps support the app.
Where are my stories stored?
All your stories are stored locally on your device using Core Data. This means:
  • Your stories are private and secure
  • No internet connection required to view saved stories
  • Deleting the app will permanently remove all stories
  • We don't have access to your stories
How do I set up image generation?
StorySpark AI supports two image generation providers:
  • Cloudflare Workers AI (Recommended): Free tier includes 10,000 neurons/day (~800-1,200 images/day). Set up in Settings → Image Generation → Cloudflare API Settings.
  • Hugging Face API: Limited free tier. Configure in Settings → Image Generation → Hugging Face API Settings.
For detailed setup instructions, check the app's README or contact support.
Can I export my stories?
Yes! You can export any completed story as a PDF. Watch a rewarded ad to unlock PDF export. The exported PDF includes all pages with illustrations and text, perfect for printing or sharing.
What is "Kids Mode" (Limit Ads)?
Kids Mode is a setting that disables interstitial ads (full-screen ads). When enabled:
  • Interstitial ads are disabled
  • Only rewarded ads are shown (with your consent)
  • Perfect for children using the app
Enable it in Settings → Ads & Privacy → Limit Ads (Kids Mode).
My story generation is stuck or failed. What should I do?
If generation fails:
  • Check your internet connection
  • Verify your API credentials in Settings (if using custom API)
  • Try regenerating the specific page using the "Retry" button
  • Make sure you have available API credits/quota
If issues persist, contact support with details about the error.
Can I use the app offline?
You can view and read saved stories offline. However, generating new stories or illustrations requires an internet connection to access the AI image generation services.

Privacy
